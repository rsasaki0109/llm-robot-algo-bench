# Security

This repository is a **local research / benchmark CLI**. It runs Python code you supply (including generated code under `tasks/generated/`).

**Do not** feed secrets, live production robot interfaces, or untrusted paths without reviewing what will execute.

To report a security issue in **this repository’s code** (e.g. unsafe defaults in the CLI), please open a **private security advisory** on GitHub if available, or email the repository owner. Please avoid posting exploit details in public issues until addressed.
